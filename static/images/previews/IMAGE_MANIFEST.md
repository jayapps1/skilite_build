# Skilite Build Premium Website Preview Image Manifest

This document catalogs every local visual asset acquired, optimized, and saved under `static/images/previews/`.

## General Directory Structure
All images are compressed in **WebP** format and optimized to be highly responsive, lightweight, and offline-compatible:
- `static/images/previews/<category_slug>/`
  - `hero_desktop.webp` (1600x900) - For desktop display sizes.
  - `hero_mobile.webp` (900x1200) - For tablet/mobile portrait display sizes.
  - `about.webp` (1200x700) - Detailed narrative company layout.
  - `product_1.webp` to `product_6.webp` (800x600) - Product cards or menu listings.
  - `gallery_1.webp` to `gallery_3.webp` (800x600) - Portfolio works items.
  - `team_1.webp` to `team_3.webp` (500x500) - Executive and practitioner team avatars.
  - `testimonial_1.webp` to `testimonial_3.webp` (400x400) - Client review circular photos.
  - `contact.webp` (1200x700) - Map or main building entrance visual.
  - `cta.webp` (1200x700) - Call to Action visual layout.
  - `cover.webp` (1200x450) - Profile page banner backdrop.
  - `logo.webp` (200x200) - Brand badge overlay icon.
